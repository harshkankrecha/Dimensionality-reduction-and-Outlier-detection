# PCA, LLE & Outlier Detection

Implementation of PCA, LLE & Outlier Detection from scratch using Python and NumPy as part of the Machine Learning Lab Course at TU Berlin.

This repository focuses on building fundamental ML algorithms without relying on high-level machine learning frameworks, with emphasis on mathematical understanding, numerical optimization, and data visualization.

---

## Implemented Algorithms

### Principal Component Analysis (PCA)
- Covariance matrix computation
- Eigen decomposition
- Dimensionality reduction
- Projection and reconstruction
- Image denoising using principal components

### Local Linear Embedding (LLE)
- k-Nearest Neighbor graph construction
- Epsilon-ball neighborhood graph
- Low-dimensional manifold embedding
- Robust graph validation and connectivity checks

### Outlier Detection
- γ-index based anomaly detection
- Distance-to-mean outlier scoring
- Comparative analysis under different contamination rates

### Evaluation Metrics
- ROC Curve generation
- AUC (Area Under Curve) implementation from scratch
